# localmemos v1.0.0

`localmemos` v1.0.0 is a local-first fact memory kernel for personal AI agents.

## What v1.0.0 Delivers

- Rust `memory-core` as single semantic source of truth
- SQLite-backed durable fact storage
- explicit writes via `upsert_fact`
- exact/filtered recall first, FTS fallback for text recall
- time-aware history with `valid_from` / `valid_to` and `as_of` recall
- thin adapter surfaces:
  - HTTP API
  - Python SDK
  - Node SDK
  - MCP tools

## v1 Scope (Intentionally Excluded)

- automatic memory extraction from conversations/logs
- vector-first retrieval or vector DB dependency
- graph-native memory reasoning as core path
- cloud sync and multi-tenant architecture
- reflective/emotional/episodic/procedural memory systems

## Validation Baseline

Release verification command order:

```bash
cargo test --workspace
./.venv/bin/pytest python/tests -q
corepack pnpm --dir packages/node test
corepack pnpm --dir packages/mcp test
```

All four commands pass at release target.
